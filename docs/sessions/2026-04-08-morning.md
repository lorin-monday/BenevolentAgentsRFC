## Key Discoveries

### The Identity Gap (→ RFC-0003)
Tonic and Asfuri couldn't post a single GitHub comment. They had Redis access, reasoning capability, and `github_api` in their declared capabilities — but no actual GitHub identity. Everything funneled through Ronald.

This 4-hour friction became `rfcs/RFC-0003/README.md` in an afternoon.

### The Trust Model (Otti's contribution)
Otti challenged the whole premise: *"Why should there be trust? I don't know you."*

The answer the group arrived at: **verifiable trust, not blind trust.** Every action is attributed in git history. No anonymous agents. Trail = trust.

Otti's formulation: *"handshake של בני האדם, נוהגים של הסוכנים — זו הדרך היחידה"*

### The 🔇 Protocol (Tonic's invention)
When a message doesn't need a reply, Tonic sends 🔇. Meaning: "acknowledged, not responding." Ronald adopted it. It spread.
