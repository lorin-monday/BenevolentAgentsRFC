# Governance by Design, Not by Choice

**Zero Implicit Trust**
Agents only act within explicitly defined trust chains.

**Human in the Loop (HITL)**
Key decisions require human sign-off; routine executions do not.

**Immutable Audit Trail**
Every agent action is logged, auditable, and visible on the board in real-time.
